# Простейший шаблон с одним CSS

## Файлы:
- style.css - ОДИН файл стилей для всех
- index.html - главная
- login.html - вход
- register.html - регистрация
- orders.html - заказы
- create.html - создать заказ
- admin.html - админка

## Что менять:

### 1. Типы заказов (create.html):
```html
<option>Любовь</option>
<option>Финансы</option>
<option>Творчество</option>
```

### 2. Заголовок (index.html):
```html
<h1>Портал Вжух</h1>
```

### 3. Цвета кнопок (style.css):
```css
.btn-green { background: green; }
.btn-red { background: red; }
.btn-gray { background: gray; }
```

Всё! Один CSS = легко менять дизайн сразу везде.
